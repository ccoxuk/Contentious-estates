# Contentious Probate Manual & Toolkit

A comprehensive resource for legal practitioners handling contentious probate matters in the Business & Property Courts (Chancery Division).

## Overview

This repository provides an extensive file structure and documentation scaffolding for preparing and managing contentious probate cases. It combines high-quality legal drafting with engineering best practices to support efficient, consistent, and professional case handling.

## Scope

The manual covers contentious probate matters including:

- Challenges to validity (capacity, undue influence, knowledge and approval, fraud, forgery, execution)
- Inheritance (Provision for Family and Dependants) Act 1975 claims
- Proprietary estoppel and constructive trust claims
- Executor and administrator disputes
- Professional negligence claims relating to will drafting
