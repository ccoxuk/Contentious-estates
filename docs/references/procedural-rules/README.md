# Procedural Rules

This directory contains procedural rules governing contentious probate proceedings in England and Wales.

## Key Rules

### Civil Procedure Rules (CPR)
- **Part 57** - Probate, Inheritance and Presumption of Death
  - The primary procedural rules for contentious probate claims
  - Covers probate actions, claims under the Inheritance Act 1975
  - Rectification and construction of wills
  - Presumption of death claims

### Related CPR Parts
- **Part 8** - Alternative Procedure for Claims (often used for probate matters)
- **Part 44** - Costs
- **Part 52** - Appeals
- Other relevant procedural rules

## File Naming

Follow the standard convention:
```
YYYYMMDD-cpr-part-number-source-v01.pdf
```

Examples:
```
20251116-cpr-part-57-justice-gov-uk-v01.pdf
20251116-cpr-part-8-justice-gov-uk-v01.pdf
```

## Metadata

Each rules document must have a corresponding metadata file in the `metadata/` directory.

## Updates

CPR rules are updated regularly. When downloading updates:
1. Increment the version number
2. Note the amendment date in the metadata
3. Keep previous versions for historical reference if significant
