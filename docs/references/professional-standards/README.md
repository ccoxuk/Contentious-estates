# Professional Standards

This directory contains professional conduct and regulatory standards relevant to contentious probate practice.

## Types of Standards

### Solicitor Regulation
- **SRA Standards and Regulations**
  - Code of Conduct for Solicitors
  - Code of Conduct for Firms
  - Accounts Rules
  - Transparency Rules

### Barrister Regulation
- **Bar Standards Board Handbook**
  - Code of Conduct
  - Conduct rules for barristers

### Costs Standards
- **Costs regulations and guidance**
  - Solicitors' costs information and client care requirements
  - Assessment of costs rules

### Continuing Professional Development
- SRA CPD requirements
- BSB CPD requirements

## File Naming

Follow the standard convention:
```
YYYYMMDD-standard-name-authority-v01.pdf
```

Examples:
```
20251116-sra-code-of-conduct-solicitors-v01.pdf
20251116-bsb-handbook-v01.pdf
```

## Metadata

Each standards document must have a corresponding metadata file in the `metadata/` directory including:
- Regulatory authority
- Date of publication/effect
- Version information
- Relevant sections for contentious probate
- Updates and amendments

## Updates

Professional standards are updated regularly. Monitor regulatory authority websites for:
- New editions
- Amendments
- Transitional provisions
- Enforcement updates
