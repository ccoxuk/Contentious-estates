# Legislation

This directory contains primary and secondary legislation relevant to contentious probate matters in England and Wales.

## Key Legislation Areas

### Primary Legislation
- **Wills Act 1837** - Governs the validity of wills
- **Administration of Estates Act 1925** - Rules on intestacy and estate administration
- **Inheritance (Provision for Family and Dependants) Act 1975** - Family provision claims
- **Trustee Act 1925** - Powers and duties of trustees
- **Trustee Act 2000** - Modern trustee investment and delegation powers
- **Insolvency Act 1986** - Relevant to insolvent estates

### Secondary Legislation
- Non-Contentious Probate Rules 1987
- Other relevant statutory instruments

## File Naming

Follow the standard convention:
```
YYYYMMDD-act-name-source-v01.pdf
```

Example:
```
20251116-wills-act-1837-legislation-gov-uk-v01.pdf
```

## Metadata

Each legislation file must have a corresponding metadata file in the `metadata/` directory with the same base filename and `.md` extension.
