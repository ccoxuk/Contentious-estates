# Legislation References

This directory contains PDF copies of relevant statutes and statutory instruments.

## Key Legislation

Typical references include:

- Inheritance (Provision for Family and Dependants) Act 1975
- Administration of Estates Act 1925
- Trustee Act 1925
- Wills Act 1837
- Forfeiture Act 1982
- Estates of Deceased Persons (Forfeiture Rule and Law of Succession) Act 2011

## File Naming

Format: `statute-{title}-{year}.pdf`

Examples:
- `statute-inheritance-act-1975.pdf`
- `statute-administration-of-estates-act-1925.pdf`

## Metadata

Each PDF must have a corresponding metadata file in `/docs/references/metadata/` with the same base name.

Required metadata fields:
- `title`: Full official title
- `type`: "statute"
- `jurisdiction`: Legal jurisdiction
- `date`: Date of Royal Assent
- `citation`: Official citation (e.g., "1975 c. 63")
- `source_url`: legislation.gov.uk URL
- `filename`: PDF filename
