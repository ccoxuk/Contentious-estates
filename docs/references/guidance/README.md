# Guidance

This directory contains guidance notes, best practice documents, and interpretative materials for contentious probate practice.

## Types of Guidance

### Court Guides
- Chancery Guide
- Queen's Bench Guide (relevant sections)
- Court-specific guides

### Judicial Guidance
- Judicial College guidance
- Senior judiciary practice statements
- Protocol and procedure notes

### Professional Body Guidance
- Law Society practice notes
- Resolution guidance
- ACTAPS (Association of Contentious Trust and Probate Specialists) materials
- Other professional body publications

### Best Practice Documents
- Costs guidance
- Electronic filing guidance
- Remote hearing guidance

## File Naming

Follow the standard convention:
```
YYYYMMDD-guidance-title-source-v01.pdf
```

Examples:
```
20251116-chancery-guide-judiciary-uk-v01.pdf
20251116-probate-costs-guidance-law-society-v01.pdf
```

## Metadata

Each guidance document must have a corresponding metadata file in the `metadata/` directory including:
- Issuing authority
- Date of publication
- Version/edition information
- Status (current/superseded)
- Scope and application
