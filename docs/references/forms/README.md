# Forms

This directory contains court forms, precedents, and templates for contentious probate proceedings.

## Types of Forms

### Court Forms
- **Probate claim forms** (as specified in CPR Part 57)
- **Part 8 claim forms** (for certain probate applications)
- **Application notices**
- **Witness statement forms**
- **Costs forms**

### Precedents
- Particulars of claim templates
- Defence templates
- Witness statement templates
- Affidavit templates
- Skeleton argument templates

### Practice Forms
- Client care letters
- Retainer agreements
- Costs information templates

## File Naming

Follow the standard convention:
```
YYYYMMDD-form-identifier-source-v01.pdf
```

Examples:
```
20251116-form-n1-claim-form-hmcts-v01.pdf
20251116-form-n208-application-notice-hmcts-v01.pdf
20251116-witness-statement-template-v01.pdf
```

## Official Sources

- HM Courts & Tribunals Service (HMCTS)
- Ministry of Justice website
- Court service websites

## Metadata

Each form must have a corresponding metadata file in the `metadata/` directory including:
- Official form number (if applicable)
- Purpose and use
- Associated rules or practice directions
- Completion guidance
- Filing requirements

## Updates

Court forms are updated periodically. Check HMCTS website regularly for:
- New versions
- Discontinued forms
- Digital/online alternatives
