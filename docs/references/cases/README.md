# Cases

This directory contains important case law and judgments relating to contentious probate matters.

## Organization

Cases should be organized by topic area and chronologically. Consider creating subdirectories for:
- Testamentary capacity
- Knowledge and approval
- Due execution
- Fraud and forgery
- Undue influence
- Rectification
- Construction of wills
- Proprietary estoppel
- Family provision claims
- Costs

## Key Landmark Cases

Examples of important cases in contentious probate:
- Marley v Rawlings [2014] UKSC 2 (rectification)
- Ilott v The Blue Cross [2017] UKSC 17 (family provision)
- Gill v Woodall [2010] EWCA Civ 1430 (testamentary capacity)
- Banks v Goodfellow (1870) LR 5 QB 549 (testamentary capacity test)

## File Naming

Follow the standard convention:
```
YYYYMMDD-case-name-citation-source-v01.pdf
```

Examples:
```
20251116-marley-v-rawlings-2014-uksc-2-bailii-v01.pdf
20251116-ilott-v-blue-cross-2017-uksc-17-bailii-v01.pdf
```

## Sources

Preferred sources for case law:
- BAILII (British and Irish Legal Information Institute)
- Official court websites
- Law reports (e.g., WLR, All ER)

## Metadata

Each case must have a corresponding metadata file in the `metadata/` directory including:
- Full citation
- Court and judge(s)
- Date of judgment
- Key legal principles
- Relevance to contentious probate
- Related cases
