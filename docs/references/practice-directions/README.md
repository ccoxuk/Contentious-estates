# Practice Directions

This directory contains Practice Directions that supplement the procedural rules for contentious probate matters.

## Key Practice Directions

### Primary Practice Directions
- **Practice Direction 57** - Probate and Inheritance
  - Supplements CPR Part 57
  - Provides detailed procedural requirements
  - Covers court forms and documentation

### Related Practice Directions
- **Practice Direction 8A** - Alternative Procedure for Claims
- **Practice Direction - Pre-Action Conduct and Protocols**
- Other relevant Practice Directions

## File Naming

Follow the standard convention:
```
YYYYMMDD-pd-identifier-source-v01.pdf
```

Examples:
```
20251116-practice-direction-57-justice-gov-uk-v01.pdf
20251116-pd-8a-justice-gov-uk-v01.pdf
```

## Metadata

Each Practice Direction must have a corresponding metadata file in the `metadata/` directory.

## Updates

Practice Directions are updated periodically to reflect changes in procedure and practice. Always check for the latest version on the official courts website.
