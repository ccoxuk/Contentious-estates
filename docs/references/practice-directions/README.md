# Practice Directions

This directory contains Practice Directions relevant to contentious probate proceedings.

## Key Practice Directions

- Practice Direction 57AC - Probate Claims
- Practice Direction 8A - Alternative Procedure for Claims
- Practice Direction 44 - Costs
- Pre-Action Protocol for the Resolution of Probate and Inheritance Disputes

## File Naming

Format: `pd-{number}-{title}.pdf`

Examples:
- `pd-57ac-probate-claims.pdf`
- `pd-8a-alternative-procedure.pdf`
- `protocol-probate-inheritance-disputes.pdf`

## Metadata

Each PDF must have a corresponding metadata file in `/docs/references/metadata/`.

Required metadata fields:
- `title`: Full title
- `type`: "practice-direction"
- `jurisdiction`: "England and Wales" (typically)
- `date`: Last update date
- `citation`: Official reference (e.g., "PD 57AC")
- `source_url`: justice.gov.uk or official source
- `filename`: PDF filename

Optional fields:
- `version`: Version number
- `status`: "current", "superseded"
- `amendments`: List of recent amendments
- `related_rules`: Associated CPR parts
