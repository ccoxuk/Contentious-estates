## Physical File Organization

<!-- TODO: Add physical file management procedures -->

### File Structure and Dividers

#### Recommended Division System

<!-- TODO: Detail physical file organization -->

1. **Section A - Client and Instructions**
   - Client care and engagement letters
   - Retainer and costs agreements
   - Client correspondence
   - Attendance notes and instructions

2. **Section B - Parties and Opponents**
   - Details of all parties
   - Opponent solicitor details
   - Correspondence with opponents

3. **Section C - Pleadings and Court Documents**
   - Claim form and particulars
   - Defence and reply
   - Applications and orders
   - Court correspondence

4. **Section D - Evidence**
   - Client witness statements
   - Other witness statements
   - Expert reports
   - Documentary evidence

5. **Section E - Correspondence**
   - Chronological correspondence file
   - Important correspondence flagged

6. **Section F - Financial and Costs**
   - Costs estimates and budgets
   - Time recording summaries
   - Bills and disbursements
   - Funding documentation

7. **Section G - Research and Strategy**
   - Legal research
   - Case strategy notes
   - Counsel's advice
   - File reviews

8. **Section H - Miscellaneous**
   - Other relevant documents

## Time Recording and Financial Management

<!-- TODO: Add time recording and billing procedures -->

### Time Recording Best Practices

<!-- TODO: Add time recording standards -->

1. **Record Contemporaneously**
   - Record time as work is done
   - Don't batch-record at end of day

2. **Detailed Narratives**
   - Clear description of work done
   - Avoid generic descriptions
   - Link to specific tasks or milestones

3. **Appropriate Time Units**
   - Use minimum time units per firm policy
   - Round appropriately

4. **Activity Codes**
   - Use correct activity codes
   - Distinguish between fee earners

### Closure Checklist
